# KeepLearning
该项目是学习一些东西的代码集合，一般跟某份教程或者某本书一致。代码+详细注释，可执行。

这个代码集合托管在github上主要出于以下几个原因：
1. 备份：在github上备份，防止本机文件丢失。
2. 同步：方便在不同电脑上进行代码同步。
3. 督促：通过github附带的各种功能（如提交统计、代码统计）来督促自己不断学习。
